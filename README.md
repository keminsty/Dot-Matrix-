# Dot Matrix

This is a cool web application that takes video files and turns them into a dynamic dot matrix display in real time. It also keeps the audio in sync with the video.

It is made using HTML5, CSS3 and the JavaScript Canvas API.

---

## Features

* **Real-Time Processing:** This application takes any video you upload and turns it into a grid of dots on the screen one frame at a time.

* **Audio Synchronization:** You can still hear the audio from the video.

* **Sleek Aesthetics:** The application has a dark and modern look that is easy on the eyes.

* **Fully Responsive:** The dot matrix display adjusts to fit screen sizes.

---

## How to Use

1. Click **Select Video** to upload a video from your device. You can upload videos like mp4 or mov files.

2. Use the **Play / Pause** button to start and stop the video and audio.

3. Watch as your video turns into a grid of glowing dots before your eyes.

---

## Technical Details

The Dot Matrix application uses a method called `getImageData()` to get the color and position of each pixel in the video. It then takes this information. Turns it into a grid of dots that look like they are made up of small circles. This gives the application a look that's similar, to old LED displays.
